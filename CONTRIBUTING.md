# Contributing to Dashlet

Thank you for your interest in contributing to Dashlet! We welcome contributions from everyone.

## Code of Conduct

Please be respectful and considerate of others. Harassment or abusive behavior will not be tolerated.

## How to Contribute

1.  **Fork the repository** and create your branch from `main`.
2.  **Install dependencies**: `npm install`
3.  **Run development server**: `npm run dev`
4.  **Make your changes**. Ensure your code matches the existing style (clean Vanilla JS and SCSS).
5.  **Test your changes**. Verify that the dashboard loads and functions as expected.
6.  **Push to your fork** and submit a **Pull Request**.

## Pull Request Guidelines

-   Provide a clear description of what your changes do.
-   Link to any relevant issues.
-   Ensure your code is clean and commented where necessary.

## Reporting Issues

If you find a bug or have a feature request, please open an issue in the repository. Provide as much detail as possible to help us understand and resolve the problem.
